深圳市汉普电子技术开发有限公司（简称“汉普电子”）成立于2003年，基于产业互联网理念为全球客户提供一站式硬件设计和智能制造服务。总部位于深圳，为全球客户提供硬件方案设计、高速高密PCB设计、PCB制造、器件采购及PCBA等一站式服务，先后在北京、上海、桂林设置分公司及客服中心，是国家和深圳市高新技术企业，拥有近百项专利和著作权。

汉普电子在高速PCB信号完整性仿真和高密PCB设计、EMC电磁兼容性、DFM可制造性设计、SIP芯片封装设计等领域积累了丰富的研发设计经验，并与英特尔（Intel）等芯片原厂、芯片分销商代理商、PCB板厂、PCBA贴片加工厂等硬件上下游合作伙伴建立了紧密合作关系，结合自主研发的信息化IT系统打造出独特的研发和供应链垂直整合一站式服务，以个性制造、柔性制造和生产满足客户硬件PCBA电路板从样品、中小批量到量产的服务需求。2016年搭载INTEL X86处理器的PCBA主机板出货超过100万台。

16年来汉普电子获得全球3000多家高新技术企业客户的认可，其硬件方案设计及一站式PCB设计和制造服务已广泛应用于AI人工智能硬件、5G通信、计算机视觉、车载电子、工业控制、安防、医疗、教育、物联网、新能源、轨道交通等领域，客户包括INTEL、TI、百度、地平线、科大讯飞等国内外知名企业。

  • 通讯网络 5G通信

    1. FPGA
    2. 高性能
    3. 高功耗
    4. 高可靠性

  • 定位于人工智能语音交互及视频应用领域，根据客户实际需求开发，致力于向客户提供专业的行业定制解决方案，如远场麦克风阵列产品、AI个人助理(包含云端语音识别、语义解析、深度学习…) 应用产品等，打造面向全球的人工智能技术应用服务和产品。

  • 计算机视觉


           汉普电子研发的人脸识别技术，能够识别物体距离并进行动态模糊拍摄，还能够进行暗光人像定位拍摄和人脸检测，目前，我们已面向汽车、安防监控、地产、医院等领域提供多种解决方案。

  • 安防

    汉普电子不仅在高速PCB设计与SI仿真技术，以及DMS一站式设计与制造业务上积累了深厚的经验，在计算机视觉、语音交互等AI技术和产品研发上也取得了长足的发展，研发和生产服务的产品已广泛应用于图像监控、防盗、防爆与安检、楼宇对讲等安防领域。                    

    1. 高速PCB设计
    2. SI仿真
    3. 计算机视觉
    4. 智能应用

  • 医疗电子

    医疗器械精密类产品将具备更强的信息综合分析处理能力，智能化工作程度更高，同时基于其特殊性对产品的生命周期、信号精准性及质量稳定性有更高的要求，汉普电子已通过医疗器械质量管理体系认证ISO13485：2003，设计和生产产品包括起搏器、X射线设备、监测设备、医疗电源、美容仪器等医用和家用智能医疗设备。                       

    1. 智能医疗
    2. 信号精准
    3. 质量稳定

  • 工业控制

    工控机是一种加固的计算机系统。工控机具有计算机属性和特征，但是会在环境比较恶劣的环境下运行，对数据的安全性要求也更高，所以工控机的加固、防尘、防潮、防辐射等设计是决定工控机性能的重要技术。汉普凭借多年的高速设计与制造经验，已熟练掌握工控机类产品的设计与制造能力，为国内外多家行业客户的工控产品研发设计与制造提供了优质的服务。                       

    1. 数据安全
    2. 性能稳定
    3. 经验丰富
    4. 优质服务

  • 汽车电子

    汉普提供一站式的汽车电子解决方案。我们运用Intel芯片打造智能行车记录仪，具备超强性能，能进行同步轨迹记录、蓝牙一键抓拍、行车实况记录、实时路况查询、点对点导航等等功能。我们生产的产品符合国际ISO/TS16949体系认证及CQC国内外质量体系认证，严格按照IPC标准和客户需求进行开发管理，保证出货时间及质量，并满足品质要求。

    1. 质量保证
    2. 及时交付
    3. 性能提升
    4. 安全驾驶

  • 智能教育/网络视频

    作为教育信息化行业一个重要分支---录播系统行业，这几年随着教育信息化的东风正在持续不断的扩大和发展。汉普凭借十几年的高速PCB设计、工程制造经验以及强大的供应链资源，帮助多家客户完成音视频互动终端、嵌入式高清录播设备、便携式高清录播一体机、图像跟踪设备等系列产品的PCB设计、PCB制造、器件采购及PCBA服务。

    1. 音视频互动终端
    2. 嵌入式
    3. 便携式
    4. 图像跟踪

  • 轨道交通

    基于轨道交通系统信息特点，传输的语音、数据、图像等应具有高度的敏感性、低时延和可靠性，汉普依靠多年来积累的丰富的信号完整性分析，高速PCB设计，可靠性设计，以及生产制造经验，完成的各类交换、调度、通信系统设备产品已广泛得到客户的认可和好评。                      

    1. 高敏感性
    2. 低时延性
    3. 安全可靠

  • 能源/环境

    随着社会的发展与进步，能源产业变的越来越重要，同时由于能源消耗所带来的环境污染问题也越发严峻，在这种情况下，国家大力推动能源产业的转型升级，倡导绿色能源，节能减排，污染控制等。汉普在绿色能源，电力，环境控制设备行业中具有丰富的研发设计经验，与业内多家能源与环境行业客户保持了长期的紧密合作，为中国的能源产业发展和环境改善做出了应有的贡献。                        

    1. 转型升级
    2. 丰富经验
    3. 紧密合作

  • 智能语音

    定位于人工智能语音交互及视频通话应用领域，基于不断优化的远场语音识别能力、云端AI助理处理能力（云端语义解析、任务分发、深度学习……）、APP应用开发能力，致力于向客户提供专业的行业定制解决方案，可为用户针对娱乐化、商务化、智能服务化等多个应用场景打造人工智能语音交互的产品。

    1. 人工智能
    2. 语音交互
    3. 视频交互
    4. 定制开发

    汉普电子基于与Intel, Rockchip 等芯片厂商的深度合作，运用Kaby Lake、Gemini Lake、Apollo Lake、Cherry Trail、RK3399等高性能芯片打造一站式平板电脑解决方案。客户产品应用于消费、教育、安防、医疗、广告、物流等领域。从ID, 板卡设计到整机、软件服务，满足客户定制化需求。使您的产品质量更可靠，拥有更快的运行速度与更极致的操作体验，实现产品从理念到上市的高效服务。

    1. 高性能
    2. 定制化
    3. 运行更快
    4. 极致体验
